Step Details

Step 1 — Dependencies

Check each tool, show ✓/✗ inline with version. Only prompt for sudo if packages are missing. If all present, no confirmation needed.

Step 2 — OAuth Credentials

Parse rclone.conf for existing Drive remotes. If found, show masked credentials and offer reuse. If not, show Google Console links and ask for input.

Step 3 — Drive Folder

Enter existing folder ID, or create a new folder. Creation triggers early OAuth + rclone mkdir + ID retrieval. Sets flag to skip re-auth in Step 8.

Step 4 — Local Folder

Ask for path. If it exists, show item count. If not, note it will be created. Validate non-empty.

Step 5 — Config Name

Letters, numbers, hyphens. Check for collisions. Offer overwrite/rename/abort on conflict.

Step 6 — Sync Schedule

Explain what this controls (remote → local polling). Presets for 1/5/10/15 min + custom cron.

Step 7 — Summary

Show everything. List what will happen (rclone, systemd, cron). Single confirm before execution. Mention sudo requirement here.

Step 8 — Execute

Numbered sub-steps [1/6]–[6/6] with ✓/✗ result each. Errors show cause + debug command. Final status with monitor/status commands.
